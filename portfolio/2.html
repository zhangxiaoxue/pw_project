Tencent Questionnaire System

It is a system where you can create questionnaires, collect responses and manage data analyses.
In the project, I helped design and took charge for the front-end development.
I applied MVC patterns, with the assistance of backboneJS. I decided to use this front-end architecture, because almost all the operations in the editing page should be processed in only one page.
